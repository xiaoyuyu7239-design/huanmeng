# 生图提示词包 v1（人工出图 → 我方接线）

> 用法：任选生图工具（即梦/豆包/Midjourney/SD 均可），逐条粘贴提示词出图；
> 每个角色出 2-4 张备选，**丢进 `素材台账/待审图/<作品slug>/` 目录**（文件名带角色id，如 `xuan-qi-1.png`），
> 然后告诉我一声——目检、压缩、接入 cast、跑全量测试都由我完成。

## 全局规范（每条提示词都默认附带）

- **竖版 2:3**（建议 1024×1536 或更高）；**单人**，人物占画面高度 ≥60%，头部完整、面部清晰——严禁多人中远景（AI 多人必崩脸）。
- 受众为女性玩家：男性角色以气质、职业感立人，禁油腻、禁夸张肌肉展示；女性角色着装完整，**禁低胸/露腰/回眸媚视等凝视构图**。
- 通用负面词：`多人, 人群, 文字, 水印, логотип, 低分辨率, 面部模糊, 畸形手指, 暴露服装, 俯拍视角`
- 风格总基调：写实厚涂 CG 插画、电影级打光（与现有场景图同族）；各作品的年代/世界观红线见分则。

## 交付流程（你出图之后）

1. 备选图丢进 `素材台账/待审图/<slug>/`，文件名含角色 id（如 `night-ferry/a-yue-2.png`）。
2. 我逐张目检（构图/着装红线/风格协调/五官完整）→ 选定 → 压缩 webp → 移入 `公共资源/portraits/<slug>/` → 接入 story.json cast → 跑 `npm test` 全量回归。
3. 未过审的图我会说明落选原因，方便你调提示词重出。
