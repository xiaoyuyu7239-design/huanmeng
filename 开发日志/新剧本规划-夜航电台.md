# 平台第 6 部新剧本 · 三案草稿

现有阵容：第九席（现代职场直播悬疑·旗舰）/ 十五封愿望（校园悬疑）/ 青丘异闻录（古风志怪查案）/ 第七织女（都市奇幻姐妹）/ 云巅修缮录（古风修仙工程）。
约束：避开"署名/名字写回"母题（已用四次）；受众 18-30 女性；素材以现有 82 张全景 + 生图管线未接入为前提；关系用 spark/trust/boundary；选择必须写明代价。

## A案《雪窗灯》—— 年代雪夜旅店密室（阿加莎式）
- 女主：随车账房温以宁，28 岁，护送一批赈灾账册过山。
- 局：大雪封山，驿站旅店一夜死了一位旅客，七名滞留者人人有秘密。
- 母题（反题）：匿名的正当性——死者身上有一份"若公开会害死另一个人"的名单，女主最终的抉择不是揭开谁，而是**决定不揭开谁**、并为此担责。
- 恋爱：同车年轻警探（程序正义 vs 人情）、旅店哑仆（秘密最重的人）。
- 素材可行性：差。雪夜旅店/年代内景库内全无，需生图管线先行。
- 风险：密室推理对分支结构要求极高（不在场证明的状态机），工程量旗舰级。

## B案《深潜站台》—— 近未来深海科考站
- 女主：轮值维生工程师简聆，站内信号中断 72 小时，六人站里有一人不该存在。
- 母题：制度回应失败后的民间存续——总部的救援协议已判定放弃，她要带全站活着回去。
- 恋爱：声呐员（只闻其声的邻舱）、随站医生。
- 素材可行性：极差，全部场景需新产。风险：科幻美术成本最高。

## C案《夜航电台》—— 90 年代深夜电台悬疑（推荐）
- 女主：程晚星，26 岁，城市之声《夜航船》深夜档主播兼来信编辑。
- 局：一封匿名来信精准"预告"了三天后的一场失踪。她白天循线查访，夜里在直播里用只有写信人听得懂的暗语回信——整座城市都在听，但只有一个人知道她在对谁说话。
- 母题（反题，直接回应复审建议）：**匿名的正当性**——写信人必须匿名才敢说真话；反派恰恰是要"实名曝光"来信人的势力。女主最终守护的是匿名说真话的权利，与前四部"把名字写回去"构成完整对话。
- 恋爱（可攻略男性）：档案科民警沈叙（程序内帮她查线，代价是他的立案权限；边界感：不动她的信源）、电台修机师阿岳（能从磁带底噪里听出录音地点，家里有一台不该存在的窃听器——秘密线）。
- 女性伙伴：台长周姨（她与女主的冲突是"保护栏目"与"保护写信人"的两难，非雌竞）；写信人本人（全程只以信件与电波存在的女性，结局才现身）。
- 章节骨架（约 24-28 节点）：序幕·第 47 封信 → 第一章·白日查访（三条线索方向：邮戳/纸张/背景音）→ 第二章·空中暗语（直播回信机制=本作特色玩法，选择你的措辞，全城可听）→ 第三章·失踪当夜 → 终章·实名风暴（反派要求公开信源）→ 结局：守护匿名（真结局）/ 找到写信人但失去栏目 / 栏目保住但信箱死亡 / 恋爱线尾声差分。
- 素材可行性：三案最佳——城市夜景、雨夜街道、天台、演播室类全景库内有现成可复用；"声音优先"的题材天然适配未来 TTS/语音管线（来信旁白、电波底噪是声音设计的富矿）；人物立绘需求少（写信人不露脸是设计）。
- 风险：90 年代质感靠文案撑；直播回信机制需要引擎支持"同一选择的公开/私密双重后果"（现有 flags+globals 可表达）。

# 平台终审结论（2026-07-17，独立内容审校智能体）

## 排序：C ＞ B ＞ A（三案是组合拳，不是三选一）
- **C《夜航电台》立即立项**：素材唯一可行（夜景/雨街/天台/演播室库内复用，写信人不露脸是把约束做成设计）；"声音优先"为未来语音管线预埋富矿。
- **B《深潜站台》等生图管线**：母题新意最强（制度回应失败后的民间存续），管线落地后第一部就是它。
- **A《雪窗灯》体面退役**：素材与状态机工程双阻塞；其核心抉择设计（"不是揭开谁，而是决定不揭开谁并为此担责"）并入 C 的真结局机制。

## C 案定稿修正（终审四处漏洞的处置）
1. **反题铁律（写进本作圣经）**：写信人只对女主一人、在电波之外现身；公共记录里她永远匿名；真结局的胜利状态＝信箱活着且无人被命名。平台定位一句话：《十五封》是"匿名指控须署名担责"，本作是"弱者的匿名证词不容强制实名"——语境不同，两者同真。
2. **直播回信机制升格为全作脊柱**：序幕以第一次空中回信收尾、终章实名风暴在直播中解决；每次直播三个措辞、公开后果走舆论 globals、私密后果走写信人 trust；立项即建"公开/私密双重账本"；反派也在听——必须有一次看似安全的早期措辞在第三章被证明是泄露源。
3. **恋爱线修正**：沈叙的轴改为内在冲突（帮她查线＝亲手弯折他信仰的程序，代价长在他自己身上）；每个结局必须交代两位男主下落。
4. **年代管理**：90 年代保留（对受众是复古滤镜非隔阂）；场景锁死年代安全景别（演播室/档案室/雨巷/室内/天台夜雨），禁用当代天际线大全景；立项写一页"年代标记物圣经"。

## 首个里程碑：垂直切片（非全剧本）
序幕 + 第一次直播回信节点在引擎内端到端跑通：一次直播、三个措辞选项，每项同时改动公开 globals 与写信人私密 trust，文案过"双重可读性"检验。交付物同捆四份文档：双重账本、年代标记物圣经、反题铁律、节点-全景映射表（含缺口清单）。验收口径沿用四部翻新标准：禁词零命中、代价书写全覆盖、双重后果引擎内可观测。切片过审再放量写第一章。

> 平台历史教训（四部翻新复审总结）：文案从来不是短板，跨状态一致性才是——先让最险的机制在最小切口上流血完毕。
