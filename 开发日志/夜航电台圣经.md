# 《夜航电台》圣经（切片版 v1 · 2026-07-17）

> 90 年代深夜电台悬疑。母题反题：**弱者的匿名证词不容强制实名**（与《十五封愿望》"匿名指控须署名担责"语境不同，两者同真）。

## 一、反题铁律（不可违背）
1. 写信人只对女主一人、在电波之外现身；**公共记录里她永远匿名**。
2. 真结局的胜利状态＝信箱活着，且无人被命名。
3. 反派的手段是"强制实名/曝光信源"；任何结局不得把"她公开身份"写成被庆祝的高潮。
4. 《雪窗灯》遗产并入真结局机制：高潮抉择是"决定不揭开谁，并为此担责"。
